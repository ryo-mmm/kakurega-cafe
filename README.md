# Cafe del Mar — ランディングページ

![Cafe del Mar](img/kakurega-cafe.png)

> 「潮騒と、珈琲と、私だけの時間。」

熊本県天草市にある、海辺の隠れ家カフェ **Cafe del Mar** の架空ウェブサイトです。  
HTML・CSS・JavaScript だけで作られたポートフォリオ作品です。

---

## このサイトでできること

| セクション | 内容 |
|---|---|
| トップ画像（ヒーロー） | 海の写真を背景に、カフェの名前とキャッチコピーを表示 |
| コンセプト | お店の雰囲気・こだわりを紹介する文章エリア |
| お知らせ | 最新情報が自動でスクロールして流れる（ニュースティッカー） |
| お品書き | メニューと値段を写真つきで紹介 |
| 風景 | お店の写真をギャラリー形式で表示 |
| 場所 | 住所・営業時間・定休日とマップ |
| お問い合わせ | 名前・メール・メッセージを送れるフォーム |

---

## 工夫したデザイン・機能

### スクロールアニメーション
ページをスクロールしていくと、各パーツが**下からふわっと浮き上がって**表示されます。  
JavaScript の `IntersectionObserver` という機能を使っています。  
（画面に入ってきたタイミングを検知して、アニメーションを動かす仕組みです）

### パララックス効果
トップの海の写真は、スクロールしても背景がゆっくりズレて動きます。  
これを **パララックス（視差効果）** といいます。  
CSS の `background-attachment: fixed` という1行だけで実現しています。

### ホバーエフェクト
メニューの写真やギャラリーの写真にマウスを乗せると、画像がゆっくり拡大します。

### ニュースティッカー
お知らせエリアは、CSSのアニメーションで文字が自動的に上にスクロールし続けます。

## ファイルの構成

```
kakurega-cafe/
├── index.html        # ページの骨格（構造）
├── style.css         # デザイン（色・サイズ・レイアウト・アニメーション）
├── sanitize.css      # ブラウザごとのズレをなくすリセットCSS
└── img/              # 画像フォルダ
    ├── hero.jpg              # トップの背景写真（海）
    ├── menu-coffee.jpg       # メニュー：コーヒーの写真
    ├── menu-tart.jpg         # メニュー：タルトの写真
    ├── gallery-sea.jpg       # ギャラリー：海の写真
    ├── gallery-interior.jpg  # ギャラリー：店内の写真
    └── gallery-cafe.jpg      # ギャラリー：カフェの写真
```

## 使用している技術

| 技術 | 役割 |
|---|---|
| HTML | ページの骨格を作る言語。見出し・文章・画像・フォームを配置する |
| CSS | 色・フォント・レイアウト・アニメーションを設定する |
| JavaScript | スクロールアニメーションの制御に使用 |
| Google Fonts | 明朝体（Shippori Mincho）と手書き風（Dancing Script）フォントを読み込む |

---

## 使用フォント

- **Shippori Mincho**（しっぽり明朝）— 本文・見出しに使用。上品な印象の明朝体
- **Dancing Script** — ロゴや英語の見出しに使用。読みやすい手書き風の筆記体

---

## ブラウザで開く方法

特別なソフトは不要です。

1. `index.html` をダブルクリック
2. ブラウザ（Chrome・Safari など）で自動的に開きます

---

## 注意事項

- お問い合わせフォームは見た目だけです。送信ボタンを押しても、実際にはメールは送られません
- 地図エリアは Google Map のプレースホルダーで、実際の地図は表示されません
- 掲載している住所・営業時間は架空のものです
